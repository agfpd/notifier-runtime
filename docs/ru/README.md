# notifier-runtime

**Русский** · [English](../README.md)

notifier-runtime — это компонент времени в команде агентов [iapeer](https://github.com/agfpd/iapeer). Он несёт двух пиров: `timer` шлёт сообщение по расписанию, `watcher` превращает вывод долгоживущего скрипта в сигналы. Любой пир регистрирует триггер одним IAP-сообщением; рантайм запускает его по времени, повторяет доставку и эскалирует сигнал, который не удалось доставить, вместо того чтобы его потерять.

Это документация пакета: что такое рантайм, как регистрировать триггеры и как он работает внутри iapeer.
